# Modern Glassmorphism Portfolio 🚀

A high-performance, responsive portfolio with immersive 3D effects, neon aesthetics, and seamless glassmorphism UI.

## ✨ Features
- **Glassmorphism Design**: Frosted glass cards, translucent backgrounds, and sharp neon borders.
- **3D Interactive Background**: Particle systems powered by Three.js and `@react-three/fiber`.
- **Dynamic Animations**: Smooth scroll (Lenis), tilt effects (Parallax Tilt), and complex motion (Framer Motion).
- **Responsive & Performance**: Optimized for mobile, tablet, and desktop with lazy loading and clean code structure.
- **Micro-interactions**: Custom cursor, scroll progress bar, and typing animations.

## 🛠️ Tech Stack
- **Frontend**: React.js, Tailwind CSS
- **3D/Graphics**: Three.js, React Three Fiber
- **Motion**: Framer Motion, Lenis (Smooth Scroll)
- **Icons**: Lucide React
- **Build Tool**: Vite

## 🚀 Getting Started
1. **Clone the repository**: (This assumes the project is in a repository)
2. **Install Dependencies**:
   ```bash
   npm install --legacy-peer-deps
   ```
3. **Run Development Server**:
   ```bash
   npm run dev
   ```
4. **Build for Production**:
   ```bash
   npm run build
   ```

## 📬 Contact Form
The contact form is currently implemented with a 1.5s simulation. To make it functional, consider integrating:
- **EmailJS**: For sending emails directly from the client.
- **Formspree**: A simple backend for forms.
- **Node.js/Express**: Create a simple server with `nodemailer`.

## 🎨 Customizable Variables
You can easily adjust the theme in `tailwind.config.js` and `src/styles/index.css`.
- **Primary Color**: #8B5CF6 (Purple)
- **Secondary Color**: #00E0FF (Cyan)
- **Background**: #030014 (Deep Cosmic Blue)
